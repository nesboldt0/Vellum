# Vellum
Welcome to Vellum, an introductory coding language that you can type right in a browser!

Vellum is a passion project of mine to introduce students to writing thier own code

Vellum is a general purpose language that features basic declaration keys

The purpose ov Vellum is to be a simple beginner language to introduce kids to the concepts of variables, functions, etc, etc.
For those curious let me explain how this all works
- User codes in Vellum
- The website reads what the user is typing and translates it to JavaScript
- When the user clicks "run code" the JS program is ran, showing the user the output of what they created

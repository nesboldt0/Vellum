# Vellum
Welcome to Vellum, an introductory coding language that you can type right in a browser!

Vellum is a project that I started as a small passion project to do in my free time

It doensn't specialize in "one thing" this is a general purpose coding language
I have 0 intentions of this going very far, this is just something for me to work on when I am bored, as well as learn how coding languages work to fully acknowlege full-fledged languages
If I had to assign any purpose to this langauge it would be as a simple beginner language to introduce kids to the concepts of variables, functions, etc, etc.
For those curious let me explain how this all works
- User codes in Vellum
- Python program analyses Vellum and translates it to JavaScript
- When the user clicks "run code" the JavaScript program is ran, showing the user the output of what they created

I am more than happy to answer any questions or take any suggestions
